# Grappling Armor

Sanabi-inspired Fabric mod for Minecraft 26.1.2. It adds grappling hooks, a grappling device, a grappling suit, range upgrades, loot/trade progression, a guidebook, advancements, sounds, textures, and chain-style hook effects.

## Features

- Grappling hook projectile with chain rope particles and a hook tip effect
- Grappling device and wearable grappling suit
- Suit right-click hook firing
- Ctrl/sprint swing assist and Shift/sneak dash assist while grappling
- Anvil range upgrades using grappling parts
- Custom creative inventory tab
- Village, shipwreck, buried treasure, and woodland mansion loot
- Master weaponsmith grappling part trade
- In-game guidebook
- Basic translations for 10 Minecraft languages

## Requirements

- Minecraft 26.1.2
- Fabric Loader 0.19.2 or newer
- Fabric API for Minecraft 26.1.2
- Java 25 or newer

## Build

Run:

```bat
build-release.bat
```

The distributable jar is copied to `dist/grappling-armor-<version>.jar`.

## Distribution

Upload this file for normal players:

```text
dist/grappling-armor-1.0.0.jar
```

Do not upload the `sources.jar` unless you specifically want to provide source artifacts for developers.

## Installation

1. Install Fabric Loader 0.19.2 or newer.
2. Install Fabric API for Minecraft 26.1.2.
3. Put `grappling-armor-1.0.0.jar` into the `.minecraft/mods` folder.
4. Launch the game.
